COMO CONFIGURAR O ARQUIVO .env:

1. Obtenha sua API Key:
   - Acesse: https://makersuite.google.com/app/apikey
   - Faça login com sua conta Google
   - Crie uma nova API key
   - Copie a chave gerada

2. Crie o arquivo .env:
   - Na mesma pasta do script principal
   - Nome exato: .env (sem extensão)
   - Cole sua API key no formato: GOOGLE_API_KEY=sua_chave_aqui

3. Estrutura final dos arquivos:
   projeto/
   ├── app.py              # Script principal do Streamlit
   ├── .env                # Suas chaves (NÃO versionar)
   ├── .env.example        # Exemplo para outros devs
   ├── requirements.txt    # Dependências
   ├── .gitignore         # Arquivos a ignorar no Git
   └── README.md          # Documentação

4. Execute:
   pip install -r requirements.txt
   streamlit run app.py

SEGURANÇA:
- Nunca commite o arquivo .env no Git
- Use sempre .env.example como template
- Mantenha suas chaves privadas
- Considere usar variáveis de ambiente em produção
